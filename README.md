
# vue-label-me
图片标注网站，用fabric.js基于 canvas对图片进行标注，包括图形矩形、圆形、多边形、直线、线段、点标记车道线，障碍物，交通信号灯等

# auto-label-web

## Project setup
```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```
